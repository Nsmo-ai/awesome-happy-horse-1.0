一個圍繞 [Happy Horse AI](https://tryhappyhorse.com/) 的公開資料與情報倉庫，重點是讓資訊更容易被驗證、理解與重用。

## 這個倉庫能提供什麼

- 目前可直接支持的事實
- 帶有 `Verified / Likely / Rumor / Outdated` 標記的說法台帳
- [Happy Horse 1.0](https://tryhappyhorse.com/) 為何突然受到關注的時間線
- 公開 benchmark 的實用解讀
- 用來評估模型的 prompt 案例
- 容易誤判的 failure case 與注意點
