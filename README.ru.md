Публичный репозиторий с интеллектом и ресурсами вокруг [Happy Horse AI](https://tryhappyhorse.com/), цель которого — сделать информацию более проверяемой, понятной и пригодной для повторного использования.

## Что даёт этот репозиторий

- факты, которые сейчас можно напрямую подтвердить
- реестр claims со статусами `Verified / Likely / Rumor / Outdated`
- таймлайн того, почему [Happy Horse 1.0](https://tryhappyhorse.com/) так быстро стал заметным
- практическое чтение публичных benchmark-страниц
- prompt-кейсы для оценки модели
- failure case и предупреждения о типичных ошибках интерпретации
