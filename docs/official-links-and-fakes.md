## Suspicious or Misleading “Official” Claims

| Link or pattern | Status | Reason |
| --- | --- | --- |
| `happyhorses.io` presented as the canonical official destination | `Likely misleading` | It appears in community repos, but it is not the URL linked from the current official public GitHub README. |
| `happyhorseone.com` presented as the official source of truth | `Likely misleading` | Referenced in community resource pages, but not directly supported by the current official public GitHub repo. |
| Any link claiming “official open-source weights available now” | `Misleading until verified` | The official public GitHub repo does not currently expose model weights or inference code. |

## Safe Rule

If a link is not directly supported by the current public repo or does not match the canonical surfaced link used by this repository, do not present it to readers as a trustworthy product entry point. Treat it as `unofficial`, `suspicious`, or `unverified` until stronger evidence appears.
