Ein öffentliches Intelligence- und Ressourcen-Repository rund um [Happy Horse AI](https://tryhappyhorse.com/), strukturiert für bessere Nachvollziehbarkeit und Wiederverwendbarkeit.

## Was dieses Repository bietet

- direkt überprüfbare Fakten
- ein Claim-Register mit `Verified / Likely / Rumor / Outdated`
- eine Timeline, warum [Happy Horse 1.0](https://tryhappyhorse.com/) so schnell sichtbar wurde
- eine praktische Lesart der öffentlichen Benchmarks
- Prompt-Fälle für Evaluierung
- Failure-Cases und Vertrauenshinweise
